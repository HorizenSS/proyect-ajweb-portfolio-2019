INFRORMATION IMPORTANTE

Ce site est un projet personnel de José Aguilla pour 2019/20. Ce projet est en face de construction permanente. Je viens de finir une formation de développeur Web que j'alterne avec des différents projets en autodidacte et en plus je continue à me former en autodidacte.

Ce site portfolio est destiné premièrement pour tous les développeurs qui volent voir mes compétences acquis mises en application et pour toutes entreprises qui veulent s'intéresser à moi pour me proposer un stage ou contrat comme développeur junior.

Ce site est simplement un prototypage en HTML / CSS / jQuery. Ce qui va me permet de voir tous les enjeux d'un projet Web,analyse, planification, stratégie, charte graphique depuis son début, proprement dit la création d'uncahier de charges.

Différent aspect d'un site Web :

  • L'architecture des pages et hiérarchie d'information.
  • La parti UX, navigation du site est anticipé le besoin utilisateur
  • La partie UI, création de la maquette, des infographies et charte graphique avec les différents outils comme (Wireframe, Mockups). Un très bon exercice pour développer ma créativité autant que graphique designer.
  • La partie conception Web HTML / CSS comme les images en :hover et les animations.
  • La partie développement front JS (interaction et partie dynamique) jQuery plugin de type carrousel,slider et autres comme de fenêtres de type modal).
  • La partie Back-end comme PHP (requêtes au serveur), SQL (langage base de données), phpmyadmin (administration de la base de données).
  • CMS (wordpress, Woocommerce, Prestashop).
  • La partie fonctionnalisté: la fonctionnalité du site et ajouter des nouvelles fonctionnalités de programmation (code) nécessaires pour la amélioration du site.

    • Dans le répertoire de github vous avez un dossier appeler (img-screnshot-site), Vous pourriez-voir ainsi les images de la partie UI (Maquette du site).
